# FairLens – AI Bias Detection System

## 🚀 Overview
FairLens is an AI-powered bias detection system developed for Solution Challenge 2026. It analyzes data and identifies potential bias in AI models to promote fairness and transparency.

## 🔍 Features
- AI bias detection
- Data analysis and fairness scoring
- Visualization of bias metrics
- Real-time insights

## 🛠️ Tech Stack
- Python
- Machine Learning
- Streamlit (for UI)
- Pandas, NumPy, Scikit-learn

## 📂 Project Structure
- app.py – Main application
- model/ – Trained model files
- data/ – Dataset
- utils/ – Helper functions
